Задание:

Написать программу, которая получает в качестве входной информации некоторый текст (из файла). В программе должен быть реализован некоторый алгоритм шифрования/дешифрования (можно использовать шифрование на основе перестановки/подстановки букв). Программа должна содержать интерфейсные элементы для выбора операции шифрования или дешифрования (кнопки, меню и т.п.). Если выбрана операция "шифрование", то текст из исходного файла (имеющего название input.txt) должен быть зашифрован и сохранен в файл encrypt.txt. Если выбрана операция "дешифрование", то текст из файла encrypt.txt должен быть расшифрован и сохранен в файл decrypt.txt.
